# AWS Services Quick Reference

Этот справочник предоставляет таблицы с основными сервисами AWS, разбитыми по секциям. Для каждого сервиса приведено краткое описание и информация о применении, а также подробности, включая особенности сервиса и важные аспекты для экзамена.

Как использовать оценки глубины

3 — уделите особое внимание: настройки, лимиты, мониторинг, best practices.

2 — разберитесь с основными сценариями и интеграциями.

1 — достаточно понимать назначение и ключевые отличия.
